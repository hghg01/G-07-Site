Trabalho 1 do segundo trimestre


Instruções

Produzir um sinaleiro “Semáforo” com diversos recursos: - Controle de carros em um cruzamento, desenvolver somente um Lado do cruzamento. - Controle da passagem de pedestre tem que ser solicitar, por botão. - Sinal sonoro para informar qual sinal luminoso o semáforo está indicando. - Adicionar o controlador visual de tempo para os Pedestres e outro para os carros, para informando que o semáforo irá mudar o sinal luminoso e sonoro. Será avaliado a entrega do projeto com a suas partes, o algoritmo de montagem e a programação comentada.


Montagem


Código

            
              // C++ code 

              int on_off = 0;
              int button = 0;
              
              void setup() 
              {
              pinMode (13, OUTPUT);          
              pinMode (12, OUTPUT);
              pinMode (12, INPUT);
              digitalWrite(13, LOW);
              digitalWrite(12, LOW);
              on_off = LOW;
              }
            
              void loop() 
              {
              button = digitalRead (12); 
              if (button == HIGH) {
                if (on_off == LOW) {
                digitalWrite(13, HIGH);
                on_off= HIGH;
                } else {
                  digitalWrite(13, LOW);
                  on_off = LOW;
                }
              }
              delay(10); // Delay a little bit to improve simulation
              }
